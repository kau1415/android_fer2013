## :open_book:Introduction 

유아에게 모바일 어플리케이션을 통해 영상 동화를 보여주며 장면마다 등장인물이 현재 상황에서 느끼게 될 감정에 대한 표정을 짓게 한다.      
해당 표정을 **CNN**을 이용해 분석하고 **감정코칭**을 적용하여 정서적인 측면에서 다른 사람을 이해하는 능력인 **조망수용능력**을 기를 수 있다.
